About CVTA

CVTA stands for Corona Virus Tracker Application. It is an extracurricular project made while learning how java applications work by using different technologies.The project helped in learning how classes and objects can help in creating a application, fetching data from a source and making the required changes before displaying it. The project also showcases the power of frameworks such as Spring-boot which can make application development less hectic and Bootstrap which helped in building the user-interface.

It acts as a portal for the public to keep track of the latest cases that have been reported globally on day-to-day basis according to the scheduled refresh time. Data related to total confirmed cases, total active cases, total recovered cases, total deaths that have been reported across the globe and also the cases reported daily are displayed.

The information displayed regarding the cases have been gathered from files located in github repository of CSSEGIandData where the data is in the form of Comma Seperated values.It is a COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University. The researchers of the university maintains and updates the files on regular basis. The information related to Covid-19 have been refrenced from who.int.
